1) Overview
-----------
This zip file contains the deliverables for the "PCA for Retail Market
Entry Strategy" assignment. The analysis evaluates European cities to 
identify the most attractive markets for retail expansion, utilizing 
Principal Component Analysis (PCA) to reduce problem's dimensionality 
and eliminate correlation bias.

2) Contents
-----------
1. PCA_report.pdf (Deliverable 1)
   The main consulting report (Word converted to PDF). It includes the 
   business context, descriptive analysis, PCA results, city rankings
   and the strategic differences between standardized and unstandardized 
   data.

2. PCA_code.R (Deliverable 2)
   The fully reproducible R script. It contains clean, step-by-step 
   commented code covering all 7 tasks.

3. Executive_summary.pdf (Deliverable 3)
   A 7 slide buisness presentation for the non-technical client containing
